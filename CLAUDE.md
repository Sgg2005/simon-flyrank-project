# Project conventions

## Stack
- Frontend: HTML, CSS, Javascript & React
- Tools: Claude Code, GitHub

## Conventions
- Commit messages will follow the Conventional Commits format (examples: `feat:`, `fix:`, `docs:`, `chore:`)
- I'll keep my code neat and easy to understand
- I'll use the correct HTML tags for each part of the page
- I'll only use React when something needs to be interactive or reused in multiple places
